# 🏭 공장 유지보수 관리 시스템

공장 설비의 유지보수 및 점검 작업을 효율적으로 관리하기 위한 Todo 애플리케이션입니다.

## ✨ 주요 기능

### 📋 작업 관리
- **작업 추가/수정/삭제**: 유지보수 작업을 쉽게 등록하고 관리
- **작업 상세 정보**: 제목, 설명, 카테고리 등 상세 정보 기록
- **실시간 통계**: 전체, 대기, 진행중, 완료, 지연 작업 현황 한눈에 파악

### 🎯 우선순위 관리
- **4단계 우선순위**: 낮음, 보통, 높음, 긴급
- **색상 구분**: 우선순위별 시각적 구분으로 중요 작업 강조

### 👥 작업자 할당
- 작업별 담당자 지정
- 작업자별 필터링으로 개인 작업 조회

### 📅 마감일 관리
- 작업별 마감일 설정
- **자동 상태 표시**:
  - 지연: 마감일 초과 작업 (빨간색)
  - 오늘 마감: 당일 마감 작업 (주황색)
  - 임박: 3일 이내 마감 작업 (노란색)
- 남은 일수 자동 계산 및 표시

### 🔄 진행 상태 추적
- **3단계 상태 관리**: 대기, 진행중, 완료
- 작업 카드에서 직접 상태 변경 가능
- 완료 시 자동으로 완료 시간 기록

### 🏷️ 카테고리 분류
- 전기설비, 기계설비, 안전점검, 예방보전, 긴급수리 등
- 카테고리별 필터링으로 특정 유형 작업만 조회

### 🔍 필터 및 정렬
- **다중 필터**: 상태, 우선순위, 작업자, 카테고리
- **정렬 옵션**: 마감일, 우선순위, 생성일, 상태
- 필터 초기화 기능

### 💾 데이터 저장
- 브라우저 로컬 스토리지에 자동 저장
- 페이지 새로고침 시에도 데이터 유지

## 🚀 시작하기

### 설치
```bash
cd factory-todo
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:5173` (또는 표시된 포트) 접속

### 프로덕션 빌드
```bash
npm run build
npm run preview
```

## 📖 사용 방법

### 1. 작업 추가
1. 우측 상단의 **"+ 새 작업 추가"** 버튼 클릭
2. 작업 정보 입력:
   - 작업 제목 (필수)
   - 작업 상세 설명
   - 카테고리 선택
   - 작업자 지정
   - 우선순위 설정
   - 상태 선택
   - 마감일 설정
3. **"추가"** 버튼 클릭

### 2. 작업 수정
1. 작업 카드의 **"수정"** 버튼 클릭
2. 정보 수정 후 **"수정"** 버튼 클릭

### 3. 상태 변경
- 작업 카드의 상태 드롭다운에서 직접 변경 (대기/진행중/완료)
- 완료로 변경 시 자동으로 완료 시간 기록

### 4. 작업 삭제
- 작업 카드의 **"삭제"** 버튼 클릭
- 확인 후 삭제

### 5. 필터링 및 정렬
- 필터 바에서 상태, 우선순위, 작업자, 카테고리 선택
- 정렬 옵션에서 원하는 정렬 기준 선택
- **"필터 초기화"** 버튼으로 모든 필터 해제

## 🎨 화면 구성

### 헤더
- 애플리케이션 제목
- 새 작업 추가 버튼
- 통계 정보 (전체/대기/진행중/완료/지연)

### 필터 바
- 상태별, 우선순위별, 작업자별, 카테고리별 필터
- 정렬 옵션
- 필터 초기화 버튼

### 작업 목록
- 작업 카드 형태로 표시
- 각 카드에는:
  - 제목 및 카테고리 배지
  - 작업 설명
  - 작업자, 우선순위, 상태, 마감일 정보
  - 생성일 및 완료일 (완료된 경우)
  - 수정/삭제 버튼

## 🛠️ 기술 스택

- **프레임워크**: Svelte
- **빌드 도구**: Vite
- **상태 관리**: Svelte Store
- **데이터 저장**: LocalStorage
- **스타일링**: CSS (컴포넌트 스코프)

## 📝 커스터마이징

### 작업자 목록 변경
`src/lib/constants.js` 파일의 `ASSIGNEE_OPTIONS` 배열 수정:

```javascript
export const ASSIGNEE_OPTIONS = [
  '미할당',
  '작업자1',
  '작업자2',
  // ... 추가
];
```

### 카테고리 추가/수정
`src/lib/constants.js` 파일의 `CATEGORY_OPTIONS` 배열 수정:

```javascript
export const CATEGORY_OPTIONS = [
  '일반',
  '전기설비',
  // ... 추가
];
```

### 색상 테마 변경
각 컴포넌트의 `<style>` 섹션에서 색상 값 수정
